# Angular Todo App Testing

## Implemented Test Cases

1. Add new task
2. Add task using Enter key
3. Prevent creation of empty task
4. Complete task
5. Edit task
6. Delete task
7. Filter Active tasks
8. Filter Completed tasks
9. Verify generated task ID
10. Verify behavior after page refresh

## Found Defects

1. Task title is not updated after editing.
2. Task remains in edit mode after changing completion status.

## Run Instructions

Install dependencies:

```bash
npm install
```

Start application:

```bash
npm start
```

Run Playwright tests:

```bash
npm run test:e2e
```

Open Playwright UI:

```bash
npm run test:e2e:ui
```

## Notes

Application uses JSONPlaceholder API.

Known API behavior:
- Changes are not persisted after page refresh.
- IDs are generated by the server.
- PUT and DELETE requests return successful responses even for non-existing IDs.

## Challenges

The main challenge was testing functionality that depends on JSONPlaceholder API behavior because changes are not persisted between sessions.
